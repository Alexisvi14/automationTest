# automationTest
Small automation challenge based on **Web UI + API testing**

This project was created using  **Java**, **TestNG**, **Selenium Webdriver** and **RestAssured**.

## Tech Stack
- Java 11+
- maven
- TestNG
- Selenium Webdriver
- RestAssured

## Prerequisites

Make sure you have installed:
-**Java JDK 11 or higher**
-**Maven 3.8+**
-**Google Chrome installed**
-**ChromeDriver available in PATH**

## List of commands to run the code

-git clone https://github.com/Alexisvi14/automationTest.git

-cd automationTest

-mvn clean compile

-mvn test
